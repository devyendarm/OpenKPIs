<h2>Overview</h2>
<div class="mui-overview">Measures the rate at which users or visitors convert into completed orders. It helps in capturing the effectiveness of website or app in converting visitors to customers</div>

<div class="mui-section">
<h2>Formula</h2>
<div class="mui-code-label">SQL</div>
<div class="mui-code-block">Order Conversion Rate = (Total Orders ÷ Total Users) × 100</div>
</div>

## Implementation

<div class="mui-section">

**BI Source System**: GA4, Adobe Analytics, Adobe Customer Journey Analytics, Amplitude, BigQuery, Query Service

</div>
## Usage & Analytics

<div class="mui-section">
**Dashboard Usage**: Shown in eCommerce Funnel, Conversion, and Marketing dashboards to gauge sales funnel performance and optimize campaign efficiency.

**Segment Eligibility**: All users with a purchase event. Used to segment conversion by channel, device, or user cohort.

**Related KPIs**: Orders, Revenue, Add-to-Cart Rate, Checkout Conversion Rate, Cart Abandonment Rate
</div>
## SQL Examples

<div class="mui-section">
**SQL Query Example**

<div class="mui-code-block" data-language="sql">sql
WITH base AS (
 SELECT DATE(event_timestamp) AS event_date, 
 COUNT(DISTINCT CASE WHEN event_name = &#x27;purchase&#x27; THEN order_id END) AS orders, 
 COUNT(DISTINCT VisitorID) AS users
 FROM ecommerce_events
 GROUP BY event_date)
SELECT event_date, 
 (orders * 100.0 / users) AS order_conversion_rate
FROM base
ORDER BY event_date;</div>
</div>
## Documentation

<div class="mui-section">
**Calculation Notes**: 1️⃣ Ensure both orders and users are from the same date range.
2️⃣ Remove test traffic.
3️⃣ Segment by channel or device where applicable.
4️⃣ Keep consistency in attribution windows.

**Details**: The Order Conversion Rate reflects the proportion of sessions (or users) leading to at least one purchase. It is a top-level KPI for assessing website effectiveness, marketing efficiency, and UX performance. Ensure alignment of session counting logic between GA4 and Adobe (visit scope) and remove internal/test activity for accuracy.
